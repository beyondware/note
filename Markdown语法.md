# 常用语法

## 标题

> 共 6 级

```sh
# 一级标题
### 三级标题
###### 六级标题
```

## 段落

```sh
两个以上空格+回车
段落与段落之间：空一行
```

### 段落对齐

```
{:--}  左对齐

{--:}  右对齐

{-:-}  居中对齐
```

## 样式

### 斜体

```sh
*斜体*
_斜体_
```

### 粗体 (2个)

```sh
**粗体**
__粗体__
```

### 斜体加粗 (3个)

```sh
***斜体加粗***
___斜体加粗___
```

### 删除线（2个）

```sh
~~删除线~~
```

### 下划线

```sh
<u>下划线<u>
```

### 上标

```sh
<sup>上标<sup>
```

### 下标

```sh
<sub>下标<sub>
```

### 分割线（3个）

```sh
***

---
```

## 图片

```sh
![说明](图片链接)

![说明](图片链接 =宽 x 高)

![说明](图片链接 =百分比)

[![说明](图标)](网址链接)
```

### 多张图片(滑动)

```sh
<![说明](链接),![说明](链接),![说明](链接)>
```

## 超链接

```sh
[说明](网址链接)
[说明](网址链接){打开方式}
<超链接>
```

### 跳转

```sh
[内容](#跳转)
```

### 锚点

```sh
[内容]{#锚点}
```

## 列表

### 无序列表

```sh
* 星号

+ 加号

- 减号
```

### 有序列表

```sh
1.
2.
3.
```

### 嵌套

```
- 外层
   + 内层嵌套
   + 内层嵌套
   + 内层嵌套
- 外层
```

## 表格

```sh
|表头|表头|表头|
|:--|:-:|--:|
|单元格|单元格|单元格|
```

## 清单

```sh
- [ ] 未完成
- [x] 已完成
```

## 转义

```sh
\   反斜线
`   反引号
*   星号
_   下划线
{}  花括号
[]  方括号
()  小括号
#   井字号
+   加号
-   减号
.   英文句点
!   感叹号
```

# 代码块

## 行内代码

```sh
`行内代码`
```

## 代码片段

```sh
<pre>
  <code>代码块</code>
</pre>
```

# 公式

## 行内公式

```sh
$\ce{行内公式}$
```

## 公式块

```sh
$$块公式$$
```

# 引用

## 特别提示

```
>{特别提示}
```

## 经验之谈

```
>{经验之谈!}
```

## 注意事项

```
>{注意事项!!}
```

# 脚注

```sh
[^脚注]
[^脚注]: 脚注说明
```

## 跟随注

```
根据勾股定理{1[通常称为毕达哥拉斯定理]}，我们可以得出三角形的面积公式。
```

## 脚注

```
根据牛顿第一定律{![又称为惯性定律]}，我们可以得出结论。
```

## 尾注

```
根据牛顿第二定律{!![又称为作用与反作用定律]}，我们可以得出结论。
```

# 编注

```
//两个斜线开头的段落为编注，会议非常特殊的样式显示，用于标注一些编辑过程中需要记录的注释，在最终的文稿中因该都去掉。
```
