# .zip

## 压缩

```sh
zip xxx.zip 被压缩单个或多个     ##文件、目录都可以
```

## 解压

```sh
unzip xxx.zip -d 指定解压路径    ##-d：为可选项
```

# .tar.gz

## 解压

```sh
tar -zxvf xxx.tar.gz
```

# .tar.bz2

## 解压

```sh
tar -jxvf xxx.tar.bz2
```

# .tar.xz

## 解压

```sh
tar -Jxvf xxx.tar.xz
```
